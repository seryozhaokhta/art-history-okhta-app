
Зависимости от угла и итерации
color c = colors[(corner + i) % 4];



Зависимость только от угла:
color c = colors[corner % 4];



Зависимость только от итерации:
color c = colors[i % 4];



Зависимость от суммы угла и итерации, умноженной на некоторый коэффициент:
color c = colors[(corner + int(i * 0.5)) % 4];



Зависимость от произведения угла и итерации:
color c = colors[(corner * i) % 4];



Зависимость от четности/нечетности суммы угла и итерации:
color c = colors[(corner + i) % 2 == 0 ? 0 : 1];
